UFOCR (User-Friendly OCR). It is YAGF fork: https://github.com/andrei-b/YAGF



*Dependencies:*
*  Qt5
*  Aspell
*  OpenCV

*How to build:*
1) Go to the directory with source code
2) Make directory for build:
```
    mkdir build
```
3) Run cmake:
```
    cmake ..
```    
4) Run make:
```
    make
```
